# ☸️ ARYAVARTA | The Celestial Archive

**Aryavarta** is a high-performance, interactive digital portal dedicated to the eternal legacy of Ancient Bharat. It bridges the gap between **Ancient Wisdom** and **Modern Web Creativity** through a seamless, single-page experience.

---

## 🌟 The Philosophy
"Simple Frontend, Extreme Creativity." 

The goal of this project was to prove that you don't need a complex backend to tell a deep story. By utilizing **HTML5**, **CSS3 (Glassmorphism)**, and **p5.js**, I have created an immersive "Akasha" (Ether) where history, mythology, and science converge.

---

## 🚀 Key Features

### 🌌 **Interactive Celestial Engine**
Powered by **p5.js**, the background is a living constellation system. Particles represent *Nakshatras* (stars) that dynamically connect as they move, creating a sacred geometry effect that responds to the user.

### ☸️ **Dharma Wisdom Wheel**
A built-in fact engine containing **50+ researched historical truths**—from the origins of surgery ($600$ BCE) to the calculation of $\pi$ by Aryabhata.

### 🔢 **Vigyan & Strategy Layers**
* **Vedic Mathematics:** Interactive cards explaining the conceptual birth of Zero ($0$) and Infinity ($\infty$).
* **Chanakya Neeti:** A deep-dive into the *Arthashastra* and the Saptanga theory of statecraft.

### 🌙 **Nakshatra Calculator**
A custom JavaScript tool that calculates a user's symbolic Moon Mansion based on Vedic Astrology principles.

### 📱 **Responsive Glassmorphism UI**
A modern, mobile-first design using blurred overlays and gold-accented borders to give a premium, "Ancient-Future" aesthetic.

---

## 🛠️ Built With

* **HTML5:** Semantic structure for SEO and accessibility.
* **CSS3:** Custom animations, Flexbox/Grid layouts, and Glassmorphism effects.
* **JavaScript (Vanilla):** Dynamic content rendering engine and search logic.
* **p5.js:** Mathematical particle simulations for the celestial background.
* **Google Fonts:** *Cinzel* for headers (Stone-carved feel) and *Lora* for body (Manuscript feel).

## 📖 How to Use

1. **Clone the Repo:** `git clone https://github.com/yourusername/aryavarta.git`
2. **Open `index.html`:** No local server or `npm install` required. It runs instantly in any modern browser.
3. **Explore:** Use the navigation bar to switch between views. Every section is rendered dynamically without a page reload.
